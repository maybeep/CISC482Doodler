# CISC482Doodler

## About
This is a simple Android drawing app. 

Features include:
- Save doodles
- Load images & past doodles
- Change brush color
- Change brush size
- Change brush opacity
- Clear drawing

## Getting Started

### Installation
1. Clone The Repo
   ```sh
   git clone https://github.com/maybeep/CISC482Doodler.git
   ```
2. Run
